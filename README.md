# Deepender Yadav — 90s Retro Portfolio

A Windows 95 / 90s retro-themed personal portfolio built with React + Vite + TypeScript.

## Stack
- React 19 + TypeScript
- Vite 8
- Tailwind CSS 3
- Pure CSS marquee animation
- Windows 95 bevel/inset borders

## Run locally
```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
```

## Live
Deploy to Vercel or Netlify — set base path to `/` in `vite.config.ts`.
